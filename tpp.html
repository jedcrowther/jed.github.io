The Perfect Pour

PHP, Bootstrap, CRUD

While there are a lot of websites that review cafes, the emphasis is often on the overall cafe experience as opposed to the quality of the coffee served. The Perfect Pour is a website where users can register an account and leave reviews about the coffee at local cafes.

In order to create a review website, it was important that the site was able to handle a variety of different modes of functionality. The website is a basic CRUD (Create, Read, Update, Delete) app built using PHP with no framework.

The site is styled using a heavily customised Bootstrap which allows the site to maintain responsiveness across different device widths so users can leave reviews while they are on-the-go.
